Follow these steps to register your portfolio with Google Search Console:

  1. Open Google Search Console
  2. Click "+ Create property"
  3. Select "URL prefix"
  4. Enter: https://www.usman2228.com
  5. Click Continue
  6. Choose verification method:
    • Recommended: DNS TXT Record
      • Copy the TXT record value Google provides
      • Go to your domain registrar's DNS settings
      • Add a new TXT record with the value
      • Come back and click "Verify"
    • Alternative: HTML File
      • Download the HTML file Google provides
      • Place it in your repository root
      • Commit and push to GitHub
      • Click "Verify"
  7. After verification, go to Sitemaps (left menu)
  8. Enter: https://www.usman2228.com/sitemap.xml
  9. Click Submit
